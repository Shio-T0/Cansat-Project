CanSat Dashboard
-----------
***QUICK DESCRIPTION***
-
  - A UI developed in python and javascript by me, from the cansat team InAxon, to better visualize the telemetry that is to be sent from the future CanSat.

  - **Contains:**
    - 4 plotly graphs for real time comparison between experimental and expected values,
    - A "most recent" image viewer + an image monitor to see all images,
    - A 3D model that will react acording to the CanSat's inclination,
    - An AI prompted with the data received for better interpretation of the telemetry,
    - Altitude, Percentual Error, Coordinates, Data Receiving Tax.
    - EVERYTHING IS UPDATED REAL TIME.
